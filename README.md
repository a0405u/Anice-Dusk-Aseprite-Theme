## Installation:
Put theme folder into `extensions` folder inside your Aseprite [preferences folder](https://www.aseprite.org/docs/preferences-folder/).

**Windows:**
```
%AppData%\Aseprite
```

**macOS:**
```
~/Library/Application Support/Aseprite
```

**Linux:**
```
~/.config/aseprite
```
